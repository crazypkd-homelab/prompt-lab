
# Documentation Prompt

## Prompt

"Write a documentation page for a custom Ansible role named `my_role`. The role has the following variables:

- `my_role_version`: The version of the software to install (e.g., "1.2.3").
- `my_role_port`: The port that the software should listen on (e.g., 8080).
- `my_role_message`: A message to be displayed by the software.

The documentation should include the following sections:
- **Description:** A brief overview of what the role does.
- **Requirements:** Any prerequisites for using the role (e.g., other Ansible roles).
- **Role Variables:** A table listing the available variables, their default values, and a description of what they do.
- **Dependencies:** A list of any other roles that this role depends on.
- **Example Playbook:** An example of how to use the role in a playbook.

Please use Markdown for formatting."
