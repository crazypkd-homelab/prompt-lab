
# Commit Message Prompt

## Prompt

"Generate a conventional commit message for the following changes:

**File:** `ansible/playbooks/webserver.yml`

**Changes:**
- Added a new task to install the `nginx` package.
- Added a new handler to restart the `nginx` service.
- Updated the `index.html` template with new content.

The commit should have a clear and concise subject line, a detailed body explaining the changes, and a footer referencing the relevant issue in the issue tracker (e.g., "Fixes #42")."
