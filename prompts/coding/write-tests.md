
# Write Tests Prompt

## Prompt

"Write a set of unit tests for the following Python function using the `pytest` framework.

```python
def add(a, b):
    return a + b
```

The tests should cover the following scenarios:
- Test with positive numbers.
- Test with negative numbers.
- Test with a mix of positive and negative numbers.
- Test with zero.
- Test with non-integer inputs (e.g., floats, strings). What should happen in this case?

Please include the necessary imports and structure the tests in a clear and organized way."
