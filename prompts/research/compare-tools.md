
# Compare Tools Prompt

## Prompt

"I'm looking for a self-hosted monitoring solution for my homelab. I've heard of a few popular options, but I'm not sure which one to choose.

Please compare and contrast the following monitoring tools:
- Zabbix
- Prometheus with Grafana
- Nagios
- Uptime Kuma

For each tool, please discuss the following:
- **Features:** What are the key features of the tool? Does it support monitoring of servers, applications, and network devices? Does it have alerting capabilities?
- **Ease of use:** How easy is the tool to set up and configure? Is it beginner-friendly?
- **Scalability:** How well does the tool scale? Can it handle a large number of devices and metrics?
- **Community and support:** How active is the community around the tool? Is there good documentation and support available?

Please provide a recommendation based on my needs. I'm looking for a solution that is easy to use, has a good UI, and can monitor a variety of devices in my homelab."
