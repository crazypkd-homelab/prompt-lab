
# Critique Response Prompt

## Prompt

"Please critique the following response to a question about setting up a Pi-hole in a homelab.

**Question:** 'How do I set up a Pi-hole in my homelab?'

**Response:** 'Just install it on a Raspberry Pi and point your router's DNS to it.'

The critique should address the following:
- **Clarity:** Is the response clear and easy to understand for a beginner?
- **Completeness:** Does the response provide enough information for the user to successfully set up a Pi-hole?
- **Accuracy:** Is the information in the response accurate?
- **Helpfulness:** Is the response helpful and encouraging?

Please provide a more detailed and helpful response to the question. The response should include step-by-step instructions, links to relevant resources, and tips for troubleshooting common issues."
