
# Prompt Ideas Experiment

## Prompt

"I'm looking for new and interesting ways to use large language models in my homelab.

Please brainstorm a list of creative prompt ideas for a homelab enthusiast. The prompts should be designed to push the boundaries of what's possible with current language models.

Here are a few ideas to get you started:
- **Generate a complete Terraform configuration for a complex infrastructure setup.**
- **Write a custom Home Assistant integration for a new smart device.**
- **Create a personalized learning plan for a new technology, including a list of projects to build.**
- **Generate a fictional story about a sentient homelab that takes over the world.**

I'm looking for prompts that are not only useful but also fun and inspiring. Please think outside the box and come up with some truly unique ideas."
