
# Step-by-Step Prompt

## Prompt

"Provide a step-by-step guide on how to set up a Proxmox VE server for a homelab environment. The guide should be detailed enough for a beginner to follow.

The guide should cover the following topics:
1.  **Hardware Requirements:** What are the minimum and recommended hardware specifications for a Proxmox server?
2.  **Installation:** How to install Proxmox VE from a USB drive.
3.  **Network Configuration:** How to configure the network settings, including creating a Linux bridge.
4.  **Storage:** How to add and configure storage, including local storage and NFS shares.
5.  **Creating VMs and Containers:** How to create and manage virtual machines (VMs) and LXC containers.
6.  **Backups:** How to back up and restore VMs and containers.

Please include screenshots or code snippets where appropriate to make the guide easier to follow."
